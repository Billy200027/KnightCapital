# KnightCapital
aplicación web Cuadros App.

“Es una aplicación privada de gestión digital de cuadros rotativos que permite administrar participantes, controlar pagos semanales, validar comprobantes y automatizar notificaciones internas.”
Incluye:
Sistema de usuarios con roles
Panel administrativo completo
Gestión automática de turnos
Historial de pagos
Confirmación de comprobantes
Notificaciones inteligentes
Control de acceso global
Publicación en Play Store
Base de datos online privada
